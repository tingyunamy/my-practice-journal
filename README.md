# 🪷 我的修行手帳 — 計數累計版

新版邏輯：

```text
本週小計 = 本週打勾天數 × 每次預計數量
年度累計 = 今年所有已儲存週次的小計加總
```

例如：念佛每次 1000 聲，本週勾 5 天，本週小計就是 5000 聲。念佛 10 萬佛號進度會讀取年度累計。

所有可計數項目都比照辦理：念佛、禪修、讀經、抄經、早課、晚課、八十八佛、義工服務、共修、布施善行。
